## вңҰ Features

### в–ҚBar
> A minimal bar with clean workspace styling, Omarchy status modules, and performance stats. ThemeвҖ‘aware, lightweight, and designed to stay out of the way while still being informative.

- **Omarchy button** вҖ” opens the main launcher
- **Workspaces** вҖ” shows your workspace list and highlights the active one
- **Media** вҖ” displays the current track and opens the mini player card on hover *(takes colors from album cover art for a more aesthetic look)*
- **System stats** вҖ” quick glance CPU and RAM, shows a progress bar on hover
- **Tray** вҖ” for applications only
- **Notification & Control Center** вҖ” access notifications, volume sliders, DND, Wi-Fi and Bluetooth manager
- **Clock** вҖ” shows time and date; hover reveals a calendar
- **Notification indicator** вҖ” lets you know when something arrives
- **Settings** вҖ” switch bar position, bar style, launcher logo, and workspace style live from the UI
- Includes Omarchy idle locking and more

---

### в–ҚDock
> Fast minimal and functional dock.

- Main button like launcher and taskmanger always pinned
- Smart Auto hide setting
- Right click to pin app to dock

---

### в–ҚLauncher
> Fast, minimal, and keyboardвҖ‘first. Opens instantly, filters as you type, and stays clean and distractionвҖ‘free.

- Instant search across all sub-menus (Global Search)
- Inspired by the Omarchy Walker menu
- Quick sections for utilities and custom menus with dedicated keybinds
- Flatpak application support

---

### в–ҚTheme & Background Picker
> A fresh interface for switching themes and wallpapers on the fly.

- Dedicated keybinds for both
- One-row scrollable layout
- Keyboard and mouse navigation

---

### в–ҚOSD
> A new-style OSD for volume and media with a fluid visualizer running in the background.

- Dedicated keybinds for each OSD
- Media OSD shows progress bar and cover art

### в–ҚMedia Card

- Play/Pause and Forward/Backward track controls

### в–ҚClipboard & Keybinds Manager

- Dedicated keybinds for both

## рҹ“Ұ Installation

> [!WARNING]
> This setup modifies your Hyprland configuration. **Backup your configs before continuing.**
> Do **not** blindly overwrite configs unless you know what you are doing.

### 1 вҖ” Install dependencies

```bash
sudo pacman -S --needed quickshell cava cliphist wl-clipboard imagemagick jq
```

### 2 вҖ” Clone the repository

```bash
git clone https://github.com/atif-1402/anomshell.git
cd anomshell
```

### 3 вҖ” Backup existing config *(recommended)*

```bash
cp -r ~/.config/quickshell ~/.config/quickshell.backup 2>/dev/null
cp ~/.config/hypr/bindings.conf ~/.config/hypr/bindings.conf.backup 2>/dev/null
```

### 4 вҖ” Install Quickshell config

```bash
cp -r config/quickshell ~/.config/
```

### 5 вҖ” Install keybinds *(manual merge recommended)*

Open `config/hypr/bindings.conf`, copy the Quickshell binds, and **manually paste them at the end of** `~/.config/hypr/bindings.conf`.

> This avoids breaking your existing keybind setup.

### 6 вҖ” Install helper scripts

```bash
cp local/bin/* ~/.local/bin/
chmod +x ~/.local/bin/qs.sh
chmod +x ~/.local/bin/kill_omarchy_conflicts.sh.sh
```

### 7 вҖ” Start Quickshell

```bash
~/.local/bin/qs.sh
```

or use the keybind: `SUPER + SHIFT + Q`

### 8 вҖ” Start Quickshell at startup *(optional)*

```bash
cat >> ./.config/hypr/autostart.conf << 'EOF'
exec-once = ~/.local/bin/wb-kill.sh
exec-once = ~/.local/bin/qs.sh
EOF
```

---

### рҹ“қ Notes

- Ensure `~/.local/bin` is in your `$PATH`
- Restart Hyprland if binds don't work
- Manual config merging is intentional вҖ” it prevents overwriting your existing setup
- Tested on a personal Omarchy setup
